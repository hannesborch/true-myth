# Interface: RetryStatus

Information about the current retryable call status.

## Properties

### count

> **count**: `number`

The 0-indexed number of times the retryable has been called.

***

### elapsed

> **elapsed**: `number`

The total duration that has elapsed across calls. Initially 0.
